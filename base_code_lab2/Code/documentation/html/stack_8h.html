Stack* create_stack ( int  max_size)

Create and initialize an empty stack.

Parameters
max_size: max number of element that the stack will contains. This operator creates an empty stack of the required capacity.
Note
If max_size is 0, the capacity of the stack will be fixed by the implementation and is a least 32.
